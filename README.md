## 🇬🇧 English

### 📝 About the Project
This project demonstrates how to work with two-dimensional arrays in C++. The program allows the user to populate an array, display it on the screen, and sort its rows.

### ✨ Key Features
The program provides **three ways** to initialize the array:
* ⌨️ **Manually:** Values are entered by the user via the keyboard.
* 🎲 **Random Generator:** Values are automatically filled using a random number generator.
* 📌 **Constant:** Values are hardcoded as constants within the program.

**Core Functionality:**
* **Sorting:** The program successfully sorts the rows of the generated array in **ascending order**.
* **Display:** Outputs the initial and sorted array cleanly to the console.

## 🇷🇺 Русский

### 📝 О проекте
Этот проект демонстрирует работу с двумерными массивами на языке C++. Программа позволяет пользователю заполнять массив, выводить его на экран и сортировать его строки.

### ✨ Ключевые особенности
В данном коде предусмотрено **три способа** заполнения массива:
* ⌨️ **Вручную:** Значения вводятся пользователем с клавиатуры.
* 🎲 **Рандомно:** Значения генерируются автоматически с помощью генератора случайных чисел.
* 📌 **Константа:** Значения заранее заданы в коде.

**Основной функционал:**
* **Сортировка:** Программа упорядочивает строки массива по **возрастанию**.
* **Вывод на экран:** Аккуратный вывод исходного и отсортированного массива в консоль.
